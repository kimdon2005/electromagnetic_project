# Coil Lab

전자기 코일 가속기에서 구슬의 1회전 기준속도를 계산하고 영상으로 측정한 포화속도와 참고 비교하는 교육용 웹 시뮬레이터입니다.

## 주요 기능

- 질량, 코일 개수, 인덕턴스, 전류, 효율에 따른 1회전 기준속도 계산
- 초기 속도와 손실 에너지를 포함한 확장 모델
- 원형 궤도의 반지름과 반복 측정 시간을 이용한 포화속도 평균 계산
- 1회전 기준값·포화 실험값의 상대 차이율과 1회전 등가 효율 계산
- 전류, 질량, 효율 변화에 따른 속도 그래프
- 보고서에 활용할 수 있는 결과 해석 문장 자동 생성
- 입력값 브라우저 자동 저장

## 실행

```bash
npm start
```

## 계산 모델

```text
총 자기 에너지 = N × ½LI²
최종 운동에너지 = ½mv₀² + η(N × ½LI²) - Eloss
예상 속도 = √(2Efinal / m)
실험 속도 = 2πr / T
상대 차이율 = |1회전 기준속도 - 포화 실험속도| / 포화 실험속도 × 100
```

## 기본 비교 조건

웹 화면의 기본값은 분석한 장치와 영상에 맞춰 설정했습니다.

- 철구슬: 9.03 g(지름 1.3 cm 강철구로 추정)
- 코일: 6개
- 인덕턴스: 5 mH(임의 추정)
- 전류: 5 A(어댑터 최대 정격을 적용한 임의 추정)
- 변환 효율: 10%(임의 추정)
- 궤도 반지름: 8.5 cm
- 실험값: 상단 촬영 7개의 정상상태 회전 주기

이 조건에서 1회전 기준속도는 약 2.882 m/s, 영상 포화속도 평균은 약 3.208 m/s, 상대 차이율은 약 10.2%이다. 두 값은 회전 단계가 다르므로 모델 오차율로 해석할 수 없다. 인덕턴스·실제 코일 전류·효율도 제조사 측정값이 아니므로 제품 성능 보증값으로 사용하면 안 된다.

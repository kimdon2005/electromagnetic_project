# Coil Lab

전자기 코일 가속기에서 구슬의 예상 최대 속도를 계산하고 영상으로 측정한 실험값과 비교하는 교육용 웹 시뮬레이터입니다.

## 주요 기능

- 질량, 코일 개수, 인덕턴스, 전류, 효율에 따른 이론 속도 계산
- 초기 속도와 손실 에너지를 포함한 확장 모델
- 원형 궤도의 반지름과 반복 측정 시간을 이용한 실험 평균 속도 계산
- 이론값·실험값 오차율 계산과 실측 효율 역산
- 전류, 질량, 효율 변화에 따른 속도 그래프
- 보고서에 활용할 수 있는 결과 해석 문장 자동 생성
- 입력값 브라우저 자동 저장

## 실행

```bash
npm start
```

브라우저에서 `http://localhost:4173` 접속합니다. 별도의 빌드 과정이나 외부 런타임 의존성은 없습니다.

## 계산 모델

```text
총 자기 에너지 = N × ½LI²
최종 운동에너지 = ½mv₀² + η(N × ½LI²) - Eloss
예상 속도 = √(2Efinal / m)
실험 속도 = 2πr / T
오차율 = |이론 속도 - 실험 속도| / 실험 속도 × 100
```

## 실험 데이터 사용 주의

첫 화면의 10 g, 4개, 5 mH, 2 A, 10%와 한 바퀴 시간 5개는 로드맵에 제시된 **예시값**입니다. 보고서에는 반드시 실제 장치에서 측정한 질량, 전류, 인덕턴스, 궤도 반지름, 영상 시간으로 교체해야 합니다.
